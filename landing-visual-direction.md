# Landing Visual Direction

## Overview

The landing page should follow a `clean editorial tech-wellness` visual direction.

The overall impression should be:

- modern
- calm
- premium
- trustworthy
- product-focused

The page must not feel like a generic SaaS template, an aggressive promo page, or a sterile medical interface.

## Core Visual Principles

### 1. Clarity First

The visual system must help users understand the product quickly.

- the value proposition should be obvious from the first screen
- sections must be easy to scan
- typography and layout must support comprehension before decoration
- the design should reduce cognitive load, not increase it

### 2. Proof Over Decoration

Visual emphasis must support trust in the product.

- interface previews should feel like proof, not filler
- ingredient labels, score badges, and analysis snippets should reinforce credibility
- decorative elements should never distract from the product explanation
- each visual accent should support understanding, trust, or conversion

### 3. Soft Premium

The page should feel polished and premium through restraint, not through visual overload.

- use generous whitespace
- prefer soft neutral surfaces over loud backgrounds
- use subtle shadows, thin borders, and rounded corners
- make the experience feel expensive through composition and rhythm

### 4. Motion With Restraint

Animation should enhance the product story without becoming the story itself.

- transitions should feel smooth and calm
- motion should support hierarchy and reveal content progressively
- floating labels and cards may have slight movement or staggered entrance
- avoid flashy, noisy, or overly complex animation patterns

## Layout Direction

- Use spacious compositions with large breathing room between sections.
- Favor a clean editorial rhythm instead of rigid repetitive SaaS blocks.
- Introduce controlled asymmetry where it improves visual interest.
- Alternate simple content sections with stronger product proof sections.
- Keep the overall reading flow straightforward despite premium composition choices.

## Typography Direction

- Headlines should be large, concise, and visually dominant.
- The main typographic tone should feel editorial rather than technical.
- Supporting copy should remain simple, readable, and low-friction.
- Small labels, badges, and support text may be used to create interface precision.
- Text hierarchy must stay obvious on both desktop and mobile.

## Color Direction

- Base palette should be light, neutral, and soft rather than pure white.
- Primary text should use a strong dark neutral for contrast and authority.
- Accent color should be used selectively for CTA emphasis and key markers.
- Functional colors should support the scoring logic:
  - green for strong / safe
  - amber for caution
  - red for risk / undesirable composition
- Color should communicate meaning, not just decoration.

## Surface and Component Direction

- Cards and content containers should use soft rounded shapes.
- Borders should be light and intentional.
- Secondary overlays may use translucent or glass-like treatment where appropriate.
- Score pills, ingredient insights, and micro-labels should feel light, floating, and intelligent.
- UI previews should be clean and legible, with enough fidelity to create confidence.

## Imagery Direction

- Combine interface-based visuals with selected human or lifestyle imagery only where it adds emotional context.
- Product UI should remain the primary proof layer.
- Avoid using imagery that turns the page into a beauty campaign or abstract brand editorial.
- Phone mockups should be used as central conversion and credibility devices.

## Motion Direction

- Use fade, slide, and stagger reveals for section entry.
- Use subtle parallax or floating movement for score badges and ingredient callouts.
- Keep hover states responsive but minimal.
- Ensure motion never harms readability, accessibility, or mobile usability.
- Support reduced-motion behavior gracefully.

## Do

- Use large, clean headlines
- Build trust through realistic product UI
- Keep the page light, calm, and premium
- Use floating AI insights and score markers as proof elements
- Preserve strong readability and conversion clarity

## Don’t

- Don’t make the landing look like a generic startup template
- Don’t overload the page with decorative gradients or effects
- Don’t push the aesthetic too far into fashion/editorial abstraction
- Don’t rely on dark mode as the main visual foundation
- Don’t use motion that competes with the product message

## Final Style Statement

The landing should feel like a premium AI utility in the wellness and product-safety space: clean, calm, editorial, and credible. It should combine airy composition, strong typography, refined product visuals, and restrained motion to create a trustworthy and modern first impression.
